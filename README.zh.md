# AI 看书

一个免费的 PDF 阅读器，内置 AI 助手。看到不懂的地方，选中文字直接问，答案显示在侧栏，不用再复制到别的聊天软件里。

[English](README.md) · 中文

## 能做什么

- 选中任意文字，解释、翻译、总结或高亮
- 书架，可放多本 PDF，带封面和阅读进度
- 自动记住读到哪；支持书签、高亮和笔记
- 让 AI 帮你找到讲某个主题的那一页
- 可以针对选中文字、当前页或整本书提问
- 每本书的聊天记录分开保存
- 普通 / 护眼 / 深色三种主题
- 中英文界面
- 手机、电脑都能用

## 怎么用

1. 打开后点 ⚙ 填一个 Key。去 [Google AI Studio](https://aistudio.google.com/app/apikey) 免费申请一个粘进去即可（只存在你自己的浏览器里）。
2. 添加一本 PDF，开始看。
3. 选中文字，点解释 / 翻译 / 总结，或者自己打字提问。

## 说明

你的书、笔记、进度都存在自己的浏览器里，不会上传到任何服务器。AI 用的是你自己的 Google Gemini Key，免费额度日常看书足够用，模型推荐 `gemini-2.5-flash-lite`。
